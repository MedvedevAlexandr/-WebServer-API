Users
======
id
name(имя)
apprentice(фамилия)
patronymic(отчество)
login(логин)
pas_hash(хешированый пароль)
photo(фото учётной записи)
school(школа)
dob(дата рождения)
events_url(ссылка на файл с основыными событиями пользователя: смена аватарки и тд)
roles(роль: админ, директор, учитель, ученик)


schools
========
id
name(имя школы)
short_name(короткое имя школы)
director(директор школы)


klass
======
id
name(имя класса. например: 9а)
schedule_url(ссылка на файл с расписанием и дз)


student(ученик)
============
id(id одинаковый с учётной записью)
school_id(id школы)
klass_id(id класса)
appraisals_url(ссылка на файл с оценками)
events_url(ссылка на файл с событиями ученика: смена школы, смена класса, переход на следующий год и тд)


director
============
id(id одинаковый с учётной записью)
school_id(id школы)
